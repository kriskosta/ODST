# Monarch Website - System Overview

## Project Description

**Monarch Website** is a premium, highly polished website project with an emphasis on visual excellence, reliability, and performance. The project uses a Halo ODST themed template as its foundation, which will be progressively transformed into the final Monarch brand.

## Purpose

This website serves as the primary web presence for Monarch, providing:
- A visually stunning user experience
- Fast, reliable performance
- Full responsive design across all devices
- Accessibility compliance (WCAG 2.1 AA)

## Technology Stack

### Frontend
- **HTML5** - Semantic markup
- **CSS3** - Custom properties, flexbox, grid, animations
- **JavaScript (ES6+)** - Vanilla JS, no framework dependencies
- **Fonts** - Orbitron (display), Rajdhani (body) via Google Fonts

### Build Tools
- **Vite** - Fast dev server and optimized builds
- **ESLint** - JavaScript linting
- **Prettier** - Code formatting

### Testing
- **Vitest** - Unit testing with JSDOM
- **Playwright** - End-to-end testing across browsers

### CI/CD
- **GitLab CI/CD** - Automated pipeline for testing and deployment
- **Local Preview Server** - Vite preview for human testing

## Current State

The project is configured with:
- Complete Halo ODST themed static website (starting point)
- Node.js project structure with npm
- Vite build system
- Unit and E2E test suites
- GitLab CI/CD pipeline
- Local deployment scripts

## Target Transformation

The ODST theme will be replaced with Monarch branding. Key areas to update:
- Color scheme (currently ODST gold/dark)
- Typography choices
- Imagery and assets
- Content and copy
- Brand identity elements
